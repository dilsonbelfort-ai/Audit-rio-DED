Como configurar o Firebase

Siga os passos abaixo para ativar o banco de dados em nuvem gratuito (Firebase Firestore).

1

Criar projeto no Firebase

Acesse console.firebase.google.com → Criar projeto → Dê um nome (ex: ded-unifap) → Desativar Analytics (opcional) → Criar.

2

Ativar Firestore Database

No painel do projeto → Build → Firestore Database → Criar banco de dados → Modo de produção → Selecionar região (southamerica-east1).

3

Configurar regras de segurança

Firestore → Aba "Regras" → Substitua o conteúdo pelo código abaixo e clique em Publicar:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /reservas/{id} { allow read: true; allow create: true; allow update, delete: if true; } match /config/{id} { allow read, write: if true; } } }
4

Copiar configuração do app

Configurações do projeto (engrenagem) → Adicionar app → Web () → Registrar → Copie o objeto firebaseConfig.

5

Colar no arquivo HTML

Abra o arquivo auditorio_ded_unifap.html em qualquer editor de texto → Localize o bloco FIREBASE_CONFIG no início do <script> → Substitua pelos valores copiados → Salve.

6

Publicar no GitHub Pages (opcional)

Crie um repositório no GitHub → Faça upload do arquivo HTML → Vá em Settings → Pages → Source: main → O link público ficará disponível em usuario.github.io/repositorio.
